Key Features

Challenging Enemies

Fight intelligent enemies that adapt to your movements and strategies.

About This Project

The Maze Game was inspired by classic first-person dungeon crawlers and modern procedural generation techniques. This project started as a learning experience for raycasting and game mechanics and evolved into a fully playable adventure.
